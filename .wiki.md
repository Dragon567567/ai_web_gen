# Project Summary
This project is an AI-powered web application designed for code generation, utilizing Vue3 for the frontend and FastAPI for the backend. It streamlines the development process by allowing users to specify requirements and receive generated code snippets in HTML, CSS, and JavaScript. By leveraging AI capabilities, it facilitates rapid prototyping and coding, enhancing developer productivity. The integration with Atoms Cloud enhances user authentication and data management through a dedicated SDK, and the backend now utilizes the **gemini-3-pro-preview** model for improved code generation.

# Project Module Description
- **Frontend**: Built with Vue3, this module handles user interactions, displays generated code, and manages the chat interface for user inputs. It includes user authentication features via Atoms Cloud.
- **Backend**: Developed using FastAPI, this module processes requests for code generation and interacts with the AI service to return generated code based on user prompts. It now utilizes the **gemini-3-pro-preview** model for enhanced code generation capabilities.

# File Description Inventory
- **backend/api/routes.py**: Defines the API endpoints for code generation.
- **backend/main.py**: Initializes the FastAPI application and includes middleware.
- **backend/models/schemas.py**: Contains data models for requests and responses.
- **backend/requirements.txt**: Lists required Python packages for the backend.
- **backend/services/ai_service.py**: Implements logic for interacting with the AI service using **gemini-3-pro-preview** to generate code.
- **backend/services/__init__.py**: Marks the services directory as a package.
- **frontend/src/components/**: Contains Vue components for chat, code preview, and code editing.
- **frontend/src/services/api.ts**: Manages API calls to the backend for code generation.
- **frontend/src/services/atomsCloud.ts**: Integrates Atoms Cloud SDK for user authentication and project management.
- **frontend/src/types/index.ts**: Defines TypeScript interfaces for messages and generated code.
- **frontend/src/App.vue**: Main component that ties together the application structure, now includes user authentication features.
- **frontend/src/main.ts**: Entry point for the Vue application, initializing the app.
- **frontend/src/pages/AuthCallback.tsx**: Handles authentication callback from Atoms Cloud.

# Technology Stack
- **Frontend**: Vue3, TypeScript, Vite, Axios, Atoms Cloud Web SDK
- **Backend**: FastAPI, Python, Pydantic
- **AI Service**: OpenAI API / Local LLM, gemini-3-pro-preview

# Usage
1. **Install Dependencies**:
   - For the backend, navigate to the `backend` directory and run:
     ```
     pip install -r requirements.txt
     ```
   - For the frontend, navigate to the `frontend` directory and run:
     ```
     pnpm install
     ```

2. **Build and Run**:
   - Start the backend server:
     ```
     python main.py
     ```
   - Start the frontend development server:
     ```
     pnpm run dev
     ```

3. **Interact with the Application**: Access the application in your browser, where you can input prompts to generate code.
