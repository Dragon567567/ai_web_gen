# Project Summary
This project is an AI-powered web application designed for code generation, utilizing Vue3 for the frontend and FastAPI for the backend. It aims to streamline the development process by allowing users to specify requirements and receive generated code snippets in HTML, CSS, and JavaScript. The application leverages AI capabilities to facilitate rapid prototyping and coding, enhancing developer productivity.

# Project Module Description
- **Frontend**: Built with Vue3, this module handles user interactions, displays generated code, and manages the chat interface for user inputs.
- **Backend**: Developed using FastAPI, this module processes requests for code generation and interacts with the AI service to return generated code based on user prompts.

# File Description Inventory
- **backend/api/routes.py**: Defines the API endpoints for code generation.
- **backend/main.py**: Initializes the FastAPI application and includes middleware.
- **backend/models/schemas.py**: Contains data models for requests and responses.
- **backend/requirements.txt**: Lists required Python packages for the backend.
- **backend/services/ai_service.py**: Implements logic for interacting with the AI service to generate code.
- **frontend/src/components/**: Contains Vue components for chat, code preview, and code editing.
- **frontend/src/services/api.ts**: Manages API calls to the backend for code generation.
- **frontend/src/types/index.ts**: Defines TypeScript interfaces for messages and generated code.
- **frontend/src/App.vue**: Main component that ties together the application structure.
- **frontend/src/main.ts**: Entry point for the Vue application, initializing the app.

# Technology Stack
- **Frontend**: Vue3, TypeScript, Vite, Axios
- **Backend**: FastAPI, Python, Pydantic
- **AI Service**: OpenAI API / Local LLM

# Usage
1. **Install Dependencies**:
   - For the backend, navigate to the `backend` directory and run:
     ```
     pip install -r requirements.txt
     ```
   - For the frontend, navigate to the `frontend` directory and run:
     ```
     pnpm install
     ```

2. **Build and Run**:
   - Start the backend server:
     ```
     python main.py
     ```
   - Start the frontend development server:
     ```
     pnpm run dev
     ```

3. **Interact with the Application**: Access the application in your browser, where you can input prompts to generate code.
