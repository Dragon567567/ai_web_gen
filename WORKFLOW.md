## 核心 API 接口

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                              API 接口列表                                        │
└─────────────────────────────────────────────────────────────────────────────────┘

认证相关 (Auth)
─────────────
POST   /api/auth/register     注册用户
POST   /api/auth/login       用户登录

代码生成 (Code)
──────────────
POST   /api/code/generate/stream   代码生成 (SSE 流式返回)
POST   /api/code/modify           代码修改
GET    /api/code/history           获取生成历史

应用管理 (Apps)
──────────────
GET    /api/apps                   获取应用列表
POST   /api/apps                   创建应用
PUT    /api/apps/{id}              更新应用
DELETE /api/apps/{id}             删除应用

部署管理 (Deploy)
────────────────
POST   /api/deploy/deploy/stream   部署应用 (SSE 流式返回)
GET    /api/deploy/deployments     获取部署列表
DELETE /api/deploy/deployments/{id} 删除部署
POST   /api/deploy/deployments/{id}/stop   停止部署
POST   /api/deploy/deployments/{id}/start  启动部署
GET    /api/deploy/deployments/{id}/status 获取部署状态

会话管理 (Session)
─────────────────
GET    /api/session/sessions           获取会话列表
POST   /api/session/sessions          创建会话
GET    /api/session/sessions/{id}     获取会话详情
PUT    /api/session/sessions/{id}     更新会话
DELETE /api/session/sessions/{id}     删除会话
POST   /api/session/sessions/{id}/messages  添加消息
```
